# pmxSyntheticData

An R project for exploring and developing tools to simulate realistic synthetic
pharmacometric (PK/PD) datasets. This may eventually become an R package, but
the repository is intentionally lightweight for now.

## Folders

- `R/` — reusable R functions
- `scripts/` — runnable workflows and experiments
- `data/` — local input and generated data
- `references/` — papers, specifications, and notes
- `output/` — generated figures, tables, and reports

Large, sensitive, proprietary, and generated data should not be committed.
