# Withdraw the acknowledgment from [`synpmx_enable_dp_engines()`](https://iamstein.github.io/synpmx/reference/synpmx_enable_dp_engines.md)

Mainly useful for tests that exercise the unacknowledged error path
without starting a fresh session.

## Usage

``` r
synpmx_disable_dp_engines()
```

## Value

`TRUE`, invisibly.
